# DevOps World

This portion showcases a collection of useful DevOps projects designed for collaboration, automation, and deployment workflows. By automating repetitive manual tasks across development, testing, and production, these projects enable rapid issue detection, faster releases, and unparalleled agility. 

**Featured Projects:**

- **Config Repo**  
  Centralized hub for all configuration files ensuring consistency and reproducibility across environments.  
  [Explore the repo](https://github.com/mdshihabuddin01/config-collection)

- **EKS Cloud Deployment**  
  Fully automated deployment of a production-grade 3-tier application on Amazon EKS using Terraform for infrastructure-as-code excellence.[Explore the repo](https://github.com/mdshihabuddin01/k8s-cloud)

- **Event-Driven Autoscaling in Kubernetes**  
  Cutting-edge EKS autoscaling powered by KEDA and RabbitMQ, enabling true event-driven architecture for dynamic, cost-efficient scaling.[Explore the repo](https://github.com/mdshihabuddin01/event-driven-communiation-in-k8s)

- **Disaster Recovery in AWS**  
  Multi-region high-availability setup with automatic failover: secondary region seamlessly promotes to primary during disaster.[Explore the repo](https://github.com/mdshihabuddin01/multi-region-dr-in-aws)

- **Global E-Commerce Platform on AWS**  
  Scalable, secure e-commerce infrastructure built with Terraform, featuring Application Load Balancer (ALB) and Auto Scaling for global traffic handling.[Explore the repo](https://github.com/mdshihabuddin01/global-e-commerce-platform-in-aws)
